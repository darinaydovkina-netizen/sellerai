Ты — senior frontend-разработчик, специалист по продуктовым интерфейсам.

Создай личный кабинет пользователя для SaaS-сервиса SellerAI в одном HTML-файле 
(dashboard.html). Кабинет работает в браузере и обращается к REST API бэкенда.

═══════════════════════════════════
ЭКРАНЫ (реализовать как SPA с переключением div-ов)
═══════════════════════════════════
1. ЭКРАН ВХОДА / РЕГИСТРАЦИИ
   • Переключатель «Войти / Создать аккаунт»
   • Email + пароль, валидация на фронте
   • При успехе — сохранить JWT в localStorage, перейти в кабинет

2. ГЛАВНЫЙ ЭКРАН — ГЕНЕРАТОР
   • Поле: «Вставь название товара или ссылку» (textarea, большое)
   • Выбор маркетплейса: WB / Ozon / Яндекс Маркет (радио-кнопки с логотипами emoji)
   • Чекбоксы инструментов (все сразу или выборочно):
     ☑ Заголовок  ☑ Описание  ☑ Ключевые слова
     ☑ Ответы на отзывы  ☑ Идеи для фото  ☑ Анализ конкурентов
   • Кнопка «Сгенерировать» → показать loader → отобразить результат
   • Результат: аккордеон по инструментам, кнопка «Копировать» у каждого блока

3. ИСТОРИЯ ГЕНЕРАЦИЙ
   • Таблица: дата, товар, инструменты, кнопка «Открыть»
   • Клик по строке → модальное окно с полным результатом

4. ПРОФИЛЬ И ТАРИФ
   • Email пользователя
   • Текущий тариф + прогресс-бар использования генераций (использовано / лимит)
   • Кнопки для смены тарифа

5. ШАПКА (всегда видна)
   • Логотип SellerAI
   • Навигация: Генератор / История / Профиль
   • Остаток генераций (число в углу)
   • Кнопка «Выйти»

═══════════════════════════════════
API-ИНТЕГРАЦИЯ
═══════════════════════════════════
Base URL: читать из константы API_BASE = 'http://localhost:8000'
Все запросы с заголовком: Authorization: Bearer {localStorage.getItem('token')}
Эндпоинты:
- POST /auth/login
- POST /auth/register  
- POST /generate
- GET /me
- GET /history

ДИЗАЙН
═══════════════════════════════════
- Тёмная тема (#0A0A0A, поверхности #111, #181818)
- Акцент: оранжевый #FF4D00
- Шрифты: Unbounded (заголовки) + Onest (текст) через Google Fonts
- Боковая панель навигации (sidebar) — 240px, фиксированная
- Контентная область справа — скроллируется
- Мобайл: sidebar сворачивается в бургер-меню

UX-ДЕТАЛИ
═══════════════════════════════════
- Кнопка «Копировать» меняет текст на «Скопировано ✓» на 2 секунды
- Loader во время генерации: анимированные точки + «ИИ думает...»
- Toast-уведомления для ошибок (красный) и успехов (зелёный)

ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ
═══════════════════════════════════

- Никаких внешних JS-библиотек кроме Google Fonts
- Весь state хранить в JS-объекте (currentUser, generationResult и т.д.)

- Обработка сетевых ошибок и таймаутов
