═══════════════════════════════════
ЧТО ДЕЛАЕТ СЕРВИС
═══════════════════════════════════
Пользователь вводит название товара или ссылку на товар конкурента.
Система генерирует через Anthropic API (claude-sonnet-4-20250514):
1. Продающий заголовок (до 100 символов)
2. Описание товара (800–1500 символов)
3. Ключевые слова (20–40 штук, сгруппированные по частотности)
4. Ответы на отзывы (позитивный и негативный шаблон)
5. Идеи для фото (5–8 сценариев)
6. Анализ конкурентов (сильные/слабые стороны + рекомендации)
7. Рекламный текст (для поста в Telegram)

ФУНКЦИОНАЛЬНОСТЬ
═══════════════════════════════════
Авторизация:
- POST /auth/register — email + password, хеширование bcrypt
- POST /auth/login — возвращает JWT access token (24 часа)

Генерация контента:
- POST /generate — принимает {product: str, tools: list[str], marketplace: str}
- Проверяет JWT, проверяет лимит генераций по тарифу
- Вызывает Anthropic API с правильным системным промтом
- Возвращает структурированный JSON с результатами

Пользователь:
- GET /me — профиль, тариф, остаток генераций
- GET /history — последние 20 генераций

Тарифы (хранить в БД):
- starter: 50 генераций/мес, 499 ₽
- business: 200 генераций/мес, 999 ₽  
- pro: безлимит, 1990 ₽

СИСТЕМНЫЙ ПРОМТ ДЛЯ AI.PY
═══════════════════════════════════
Используй этот системный промт при каждом запросе к Anthropic API:

«Ты профессиональный маркетолог маркетплейсов. Пользователь даёт тебе название 
или ссылку на товар и выбирает нужные инструменты. Ты отвечаешь ТОЛЬКО валидным 
JSON без markdown-обёртки. Структура: {"title": "...", "description": "...", 
"keywords": {"high": [], "mid": [], "low": []}, "review_positive": "...", 
"review_negative": "...", "photo_ideas": [], "competitor_analysis": {
"strengths": [], "weaknesses": [], "recommendations": []}, "ad_text": "..."}. 
Генерируй только те поля, которые запрошены. Пиши на русском языке. 
Без воды и клише — только конкретика.»

ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ
═══════════════════════════════════
- FastAPI + uvicorn
- SQLAlchemy 2.0 async + aiosqlite
- python-jose для JWT
- passlib[bcrypt] для паролей
- anthropic (официальный SDK)
- pydantic-settings для конфига
- CORS настроен для localhost и production домена
- Все секреты через .env (ANTHROPIC_API_KEY, JWT_SECRET, DATABASE_URL)
- Обработка ошибок: 400/401/403/429/500 с понятными сообщениями
- Логирование через logging (INFO уровень)
